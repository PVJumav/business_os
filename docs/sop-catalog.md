# SOP Catalog

Business OS 5.5 seeds SOP structures for HR, Finance, Sales, Marketing, Customer Desk, Technical, Projects, Legal, Operations, and Management.

Each SOP supports:

- SOP code and title.
- Department owner.
- Owner role.
- Related policy.
- Related workflow.
- SLA hours.
- Required evidence.
- Review cycle.
- Active status.
- Step order and step-level approver role.

The SOP model is designed to let a department convert written procedures into controlled workflow stages and auditable task execution.
